\### 项目结构

\- 采用模块化设计，包含 src/app.js 主应用文件

\- 分离的视图文件： index.ejs （列表）、 add.ejs （添加）、 detail.ejs （详情）、 edit.ejs （修改）

\- 数据库模块：使用文件系统存储数据（ medicines.json ）

\- 公共资源： styles.css 样式文件

\### 技术栈

\- 后端 ：Node.js + Express

\- 前端 ：EJS模板引擎 + 原生CSS

\- 数据存储 ：文件系统（JSON格式）

\- 样式 ：极简设计，马卡龙绿主题

\### 实现的功能

1\. 药品列表展示 ：显示序号、通用名称、商品名称、相关疾病、用法用量，并提供详情、修改、删除按钮

2\. 药品添加 ：完整的表单，包含所有关键字段

3\. 药品查询 ：支持根据通用名称、商品名称、相关疾病进行模糊搜索

4\. 药品详情 ：展示所有字段信息的详情页面

5\. 药品修改 ：支持修改所有字段的修改页面

6\. 药品删除 ：带有确认对话框的删除功能

\### 运行状态

\- 数据文件已初始化，系统可以正常添加、查询、修改和删除药品信息
